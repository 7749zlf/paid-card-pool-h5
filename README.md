# 付费解锁卡池 H5

这是一个独立的静态 H5 页面，用于演示“短剧播放页 -> 付费节点 -> 未解锁卡池内容 -> 付费确认 -> 解锁展示 -> 抽取/记录/订单”的前端流程。

## 在线访问

访问地址：https://7749zlf.github.io/paid-card-pool-h5/

## 本地查看

直接打开 `index.html` 即可查看页面。

## 已包含功能

- 竖屏短剧播放页
- 播放页提供 3 个相同视频，上下滑动切换
- 视频切换采用整屏纵向轨道，支持跟手拖动、松手吸附和边界阻尼
- 视频播放到结尾时居中弹出付费节点提示并显示蒙版，弹出后视频暂停，付费节点和蒙版会跟随对应视频上下滑动
- 播放页右侧操作：静音、收藏、点赞、选集、更多
- 选集弹层，未解锁剧集会进入付费解锁流程
- 个人中心页面，包含余额、绑定邮箱、收藏、点赞、观看历史、充值记录、消费记录
- 未付费时卡池内容遮罩锁定
- 点击锁定卡片或按钮唤起支付弹层
- 支持 PayPal、银行卡、余额三种演示支付方式
- 支付确认后保存本地解锁状态
- 付费开放卡池入口；每次抽取都需要单独支付，支付成功后随机解锁 1 个内容
- 单次抽取支付成功后播放抽卡动画，再展示随机解锁的内容
- 支持查看已解锁卡牌详情
- 右上角可重置测试状态

## 接真实支付

当前支付逻辑是前端演示，不会产生真实扣款。上线接入时，把 `card-pool.js` 里的 `confirmPayment()` 替换成真实支付创建订单、跳转收银台、支付回调校验和解锁写入即可。
