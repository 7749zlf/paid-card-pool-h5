# 付费解锁卡池 H5

这是一个独立的静态 H5 页面，用于演示“短剧播放页 -> 付费节点 -> 未解锁卡池内容 -> 付费确认 -> 解锁展示 -> 抽取/记录/订单”的前端流程。

## 本地查看

直接打开 `index.html` 即可查看页面。

## 已包含功能

- 竖屏短剧播放页
- 播放页右侧操作：静音、收藏、点赞、选集、更多
- 选集弹层，未解锁剧集会进入付费解锁流程
- 未付费时卡池内容遮罩锁定
- 点击锁定卡片或按钮唤起支付弹层
- 支持 PayPal、银行卡、余额三种演示支付方式
- 支付确认后保存本地解锁状态
- 解锁后展示完整卡池、掉率、抽取记录、订单记录
- 支持单抽并查看卡牌详情
- 右上角可重置测试状态

## 接真实支付

当前支付逻辑是前端演示，不会产生真实扣款。上线接入时，把 `card-pool.js` 里的 `confirmPayment()` 替换成真实支付创建订单、跳转收银台、支付回调校验和解锁写入即可。
